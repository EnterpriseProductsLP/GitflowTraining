Feature Workflow

Start a new feature:

Development of new features starts from the "develop" branch.
git flow feature start add-feature-workflow

Work on your feature:

Work on your feature, and create one or more commits.
git add .
git commit -am "Did some work."

Finish your feature:

Finish your feature. This action:
  • Merges your feature into develop.
  • Removes the feature branch.
  • Switches back to the develop branch.
fit flow feature finish add-feature-workflow
