Release Workflow

Start a new release:

Releases start from the "develop" branch.
Optionally can be started from a [base] branch.
git flow release start [version] [base]

Prepare your release:

Make last-minute bug fixes.
git add .
git commit -am "Fixed the last bug."

Finish your release:

Finish your release. This action:
  • Merges the release branch back into "master."
  • Tags the release with its name.
  • Back-merges the release into "develop."
  • Removes the release branch.
  • Pushed the release tag.
git flow release finish [version]
git push
git push --tags
